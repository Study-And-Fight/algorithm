### 精通一个领域:

- chunk it up 切碎知识
    - 把知识繁复的体系碎片化消化  →  思维导图

- 数据结构分类：
    1. 一维        
        1. 基础:
            1. 数组 array (Ex:string)
            2. 链表 linked list
        2. 高级
            1. 栈 stack
            2. 队列 deque
            3. 双端队列 deque
            4. 集合 set
            5. 映射 map (hash or map)
    2. 二维
        1. 基础
            1. 树 tree
            2. 图 gragh
        2. 高级
            1. 二叉搜索树 binarySearchTree (red-black tree,AVL)
            2. 堆 heap
            3. 并查集 disjoint Set
            4. 字典树 Trie
        3. 特殊
            1. 位运算 Bitwise
            2. 布隆过滤器 BloomFilter
            3. LRU Cache
